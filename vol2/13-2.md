# 铁索连环 · 变长元组 tuple

`pair` 能把两个东西绑在一起。但真实世界经常一次要绑好几个——比如一次函数调用返回三个结果、或者把分散的数据打包传给另一个函数。

`std::tuple` 就是**任意个、任意类型的东西绑在一起**。在 `<tuple>` 中。

------

## 创建与解包

```cpp
#include <tuple>
#include <iostream>
#include <string>

int main()
{
    // 三条信息一次打包
    tuple<string, int, double> student("张三", 20, 95.5);

    // auto 推导更省事
    auto info = make_tuple("李四", 22, 88.0, "计算机系");

    // 取值（索引必须是编译期常量）
    cout << get<0>(student) << "\n";   // 张三
    cout << get<1>(student) << "\n";   // 20
    cout << get<2>(student) << "\n";   // 95.5

    // C++17 结构化绑定——清爽解包
    auto& [name, age, score] = student;
    cout << name << " " << age << " " << score << "\n";
}
```

`get<索引>` 的索引必须写死为编译期常量——变量不行。如果你需要按运行时条件取第几个，tuple 不适合，应该用数组或 vector。

------

## 函数返回多个值

Tuple 最常见的实战场景就是让函数一次返回多个值，而你不用为了这还专门定义一个 struct：

```cpp
#include <tuple>
#include <vector>
#include <iostream>
#include <algorithm>

// 返回最小、最大、平均——一次三个结果
tuple<int, int, double> stats(const vector<int>& data)
{
    int min = *min_element(data.begin(), data.end());
    int max = *max_element(data.begin(), data.end());
    double sum = 0;
    for (int x : data) sum += x;
    double avg = sum / data.size();
    return {min, max, avg};     // C++17 起可用初始化列表构造
}

int main()
{
    vector<int> scores = {85, 92, 78, 95, 88, 76, 99, 81};
    auto [min, max, avg] = stats(scores);
    cout << "最低: " << min << ", 最高: " << max
              << ", 平均: " << avg << "\n";
}
```

以前你要么用引用参数（把结果写回调用处），要么专门写个 struct 打包，要么用 `pair` 放两个（放不下三个就得嵌套 pair）。`tuple` + 结构化绑定把这些历史折衷一次化掉了。

------

## `tie`：引用式解包和比较

`std::tie` 创建的是一个**全部为引用**的 tuple——用于把一组已有变量的"身份"绑到 tuple 的某个位置上，然后可以一次性赋值给它们或对它们进行字典序比较：

```cpp
#include <tuple>
#include <iostream>

int main()
{
    int a = 0, b = 0;

    tie(a, b) = make_tuple(1, 2);   // 把 tuple 的两个元素分别写入 a 和 b
    cout << a << " " << b << "\n";         // 1 2

    // 常见用法——多字段字典序比较
    struct Record { int priority; string name; };

    Record r1{1, "Apple"}, r2{2, "Banana"};

    // 把 priority 和 name 揉成比较组——先比 priority，再比 name
    if (tie(r1.priority, r1.name) < tie(r2.priority, r2.name))
        cout << "r1 先于 r2\n";
}
```

`tie` 的技巧：用它替代手写多字段 `if (a.priority != b.priority ? a.priority < b.priority : a.name < b.name)` 这种层层嵌套比较，代码直降成一行的字典序 tuple 比较。

------

## tuple 的辅助工具

| 工具 | 作用 |
| ---- | ---- |
| `std::make_tuple` | 从已有值推导类型创建 tuple |
| `std::tie` | 创建"全引用型" tuple——用来解包或比较 |
| `std::tuple_cat` | 拼接两个 tuple 成一个 |
| `std::tuple_size<T>` | 编译期常量——tuple 有几个元素 |
| `std::tuple_element<N, T>` | 第 N 个位置的类型是什么 |

取舍：如果这组字段在项目中反复出现并且有明确的业务含义（比如多次返回同一个"设备状态"组合），就值得为此专门定义 struct 取代 tuple——struct 能给每一个字段起有意义的名称，提高代码的可读性。但如果组合只是一次性的——比如"这个函数需要一次返回三个值但以后不会再以这种组合出现"——tuple 是不引入新类型名的最低阻力选择。
