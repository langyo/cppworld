# 有条不紊 · 格式化输入输出

我们一直在用 `cout << "文本" << 数字 << endl;`。这很管用，但当你需要对齐表格、控制小数位数、设定字段宽度时，光靠默认格式就远远不够了。

C++ 提供了一套**流操纵符**，藏在 `<iomanip>` 头文件中。它们像遥控器般对 `cout`、`ostringstream` 等流发号施令——设定输出格式、精度、对齐等。操纵符和流搭配得天衣无缝，因为它们返回的也是流引用，可以像 `<<` 一样链式写。

------

**设置小数精度**

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

double pi = 3.14159265358979;

cout << setprecision(3) << pi << endl;      // 3.14
cout << setprecision(5) << pi << endl;      // 3.1416
cout << fixed << setprecision(2) << pi << endl; // 3.14——fixed 确保始终按小数点后位数显示
```

`setprecision` 设置的是**有效位数**还是**小数点后位数**，取决于前面有没有 `fixed`（或 `scientific`）开关：不带开关时它是总的有效位数，带 `fixed` 时它是小数位数。

| 操纵符 | 效果 |
| ------ | ---- |
| `fixed` | 固定小数点格式（3.140000） |
| `scientific` | 科学记数格式（3.140000e+00） |
| `hexfloat` | 十六进制浮点格式（极少用） |
| `defaultfloat` | 恢复默认格式 |

------

**设置字段宽度与填充**

```cpp
cout << setw(10) << "Name" << setw(5) << "Age" << endl;
cout << setw(10) << "Alice" << setw(5) << 25 << endl;
cout << setw(10) << "Bob" << setw(5) << 30 << endl;
// 输出：
//       Name  Age
//      Alice   25
//        Bob   30
```

`setw` 只对**紧接其后的一个输出**有效——输出完这个字段后自动还原。而 `setfill` 是粘性的，设一次之后所有后续的宽度填充都会沿用该字符：

```cpp
cout << setfill('-') << setw(10) << 42 << endl;  // --------42
```

------

**对齐方式**

```cpp
cout << left << setw(10) << "Name" << right << setw(5) << "Age" << endl;
// Name        Age
// (Name 左对齐 10 格，Age 右对齐 5 格)
```

`left` 和 `right` 也是粘性的——设置了之后会一直沿用，直到遇到对立的另一个对齐指令。

------

**布尔值显示为文字**

```cpp
cout << boolalpha << true << " " << false << endl;  // true false
cout << noboolalpha << true << " " << false << endl; // 1 0（默认）
```

------

**整数进制切换**

```cpp
cout << oct << 255 << endl;   // 377
cout << hex << 255 << endl;   // ff
cout << dec << 255 << endl;   // 255——恢复十进制

cout << showbase << hex << 255 << endl;     // 0xff  —— 显示进制前缀
cout << uppercase << hex << 255 << endl;    // 0XFF  —— 字母全大写
```

`oct`/`hex`/`dec` 是粘性的，`showbase` 和 `uppercase` 也是。如果想输出十六进制时前面有 `0x` 前缀，就配合 `showbase`。

------

**自定义操纵符**

你也可以自己写操纵符：

```cpp
ostream& hr(ostream& os)
{
    return os << string(40, '-') << endl;
}

int main()
{
    cout << "章节标题" << endl << hr;
    // 章节标题
    // ----------------------------------------
}
```

操纵符的本质就是一个函数——接收 `ostream&` 返回 `ostream&`。它和 `endl` 是同样的东西（`endl` 也是一个函数，插入换行并调用 flush）。

------

这些流操符让 C++ 的流输出不再只是简单的打印工具——它能做出对齐漂亮的表格、精准的小数输出、十六进制调试打印等一系列日常开发工作。以后写日志、输出报告时，别忘了 `<iomanip>` 里这群幕后的排版专家。
