# 有条不紊 · 格式化输入输出

默认的 `cout << 值;` 够用，但当你需要排表、控制小数位数、十六进制调试时，默认的输出完全不够看。

C++ 的标准库提供了一套**流操纵符（manipulators）**，存放在 `<iomanip>` 中。它们不改变被输出的数据本身，只改变流在输出时的格式表现。因为是粘合在 `<<` 链上的，用起来无比顺手。

------

## 设置小数精度

```cpp
#include <iostream>
#include <iomanip>

int main()
{
    double pi = 3.14159265358979;

    std::cout << std::setprecision(3) << pi << "\n";   // 3.14
    std::cout << std::setprecision(5) << pi << "\n";   // 3.1416
    std::cout << std::fixed << std::setprecision(2) << pi << "\n"; // 3.14

    // fixed 让 setprecision 控制"小数点后几位"而不是"有效位数"
}
```

不加 `fixed` 时 `setprecision` 作用于有效数字总位数。加了 `fixed` 后作用于小数点后几位。

| 操纵符 | 效果 |
| ------ | ---- |
| `std::fixed` | 固定小数点格式（3.140000） |
| `std::scientific` | 科学记数法（3.140000e+00） |
| `std::defaultfloat` | 复原默认格式 |

## 字段宽度、填充和对齐——造一张对齐的表

```cpp
#include <iostream>
#include <iomanip>

int main()
{
    // 左对齐的两列
    std::cout << std::left
              << std::setw(12) << "商品"
              << std::setw(8)  << "价格" << "\n"
              << std::string(20, '-') << "\n"
              << std::setw(12) << "苹果"
              << std::setw(8)  << 5.5   << "\n"
              << std::setw(12) << "香蕉"
              << std::setw(8)  << 3.8   << "\n"
              << std::setw(12) << "西瓜"
              << std::setw(8)  << 12.0  << "\n";
}
```

输出：

```
商品        价格
--------------------
苹果        5.5
香蕉        3.8
西瓜        12
```

`setw` 只对**紧接其后的一个输出**有效——输出完自动重置。而 `left`、`setfill` 是粘性的，设一次之后一直沿用。

```cpp
std::cout << std::setfill('0') << std::setw(5) << 42;   // 00042
```

## 进制切换和 bool 展示

```cpp
int x = 255;

std::cout << std::oct << x << "\n";   // 377（八进制）
std::cout << std::hex << x << "\n";   // ff（十六进制）
std::cout << std::dec << x << "\n";   // 255（切回十进制）

std::cout << std::showbase << std::uppercase;
std::cout << std::hex << x << "\n";    // 0XFF（带前缀，大写）

std::cout << std::boolalpha << true << " " << false << "\n"; // true false
```

## 写一个简易报告生成器

把上面学的拼起来——打印一张带分隔线、日期、对齐的简单报表：

```cpp
#include <iostream>
#include <iomanip>
#include <vector>
#include <string>
#include <ctime>

struct Record { std::string name; int quantity; double price; };

void printReport(const std::vector<Record>& records)
{
    auto t = std::time(nullptr);

    std::cout << "=== 销售报表 ===\n";
    std::cout << "日期: " << std::ctime(&t);
    std::cout << std::string(40, '=') << "\n";

    std::cout << std::left
              << std::setw(16) << "商品"
              << std::setw(10) << "数量"
              << std::setw(10) << "单价"
              << std::setw(10) << "总额" << "\n";
    std::cout << std::string(40, '-') << "\n";

    std::cout << std::fixed << std::setprecision(2);
    double grandTotal = 0;
    for (const auto& r : records)
    {
        double total = r.quantity * r.price;
        grandTotal += total;
        std::cout << std::setw(16) << r.name
                  << std::setw(10) << r.quantity
                  << std::setw(10) << r.price
                  << std::setw(10) << total << "\n";
    }

    std::cout << std::string(40, '-') << "\n"
              << std::setw(36) << "总计:"
              << std::setw(10) << grandTotal << "\n";
}

int main()
{
    std::vector<Record> sales = {
        {"苹果", 100, 5.5},
        {"香蕉", 80,  3.8},
        {"西瓜", 30,  12.0}
    };
    printReport(sales);
}
```
