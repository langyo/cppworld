# 换一种交流的方式 · 成员访问运算符定义

有些时候，你会希望自己的类在行为上"冒充"某个类型。比如智能指针——它外部看起来就像普通的指针，但内部做了更多事情（比如自动释放）。

这种伪装靠的也是运算符重载。两个最重要的"假身份"制造者：**解引用运算符 `*`** 和**箭头运算符 `->`**。

------

**`*` 和 `->`：扮成指针**

来看看一个最简化的 `unique_ptr` 是怎么做到"像指针"的：

```cpp
template<typename T>
class SimplePtr
{
private:
    T* raw;
public:
    explicit SimplePtr(T* p) : raw(p) {}

    ~SimplePtr() { delete raw; }

    T& operator*() const       // *ptr —— 拿到所指对象本身
    {
        return *raw;
    }

    T* operator->() const      // ptr-> —— 拿到所指对象的地址（用于访问成员）
    {
        return raw;
    }
};
```

使用起来：

```cpp
SimplePtr<Point> sp(new Point{3, 4});
cout << (*sp).x << endl;   // 3 —— 先用 * 解引用拿到 Point，再 .x 访问
cout << sp->x << endl;     // 3 —— 直接用箭头访问成员
```

`operator->` 的行为有点特别：它返回一个指针，然后编译器会自动对该指针再执行一次 `->` 操作来最终访问目标成员。在标准里这被称为 `operator->` 的"连锁解引用"机制——只要它返回的东西还能再 `->`，编译器就会继续解下去。

你可以试着构造一个"套娃"——让一个类的 `operator->` 返回另一个类的对象，那个类又有自己的 `operator->`。编译器会一串解到底，非常有意思。

------

**智能引用：`operator.` 的未来**

C++ 目前还不支持重载 `.` 运算符（不过 C++23 之后有提案在推进了）。如果你想实现和 `->` 一样的"间接访问但不需要写箭头"的效果，目前只能靠**转换运算符**（下一节讲）来变通——让对象隐式转换为内部持有的引用类型。

这实际上是 C++ 的一个历史限制。像 Rust 语言的 `Deref` trait 就能让智能指针行为在任何需要引用的地方无缝接替，而在 C++ 中要达到相同效果需要多写转换代码。

------

**`[]` 下标运算符**

如果你想让自己的类像数组一样能用下标访问，重载 `[]`：

```cpp
class VectorWrapper
{
private:
    vector<int> data;
public:
    int& operator[](size_t index)
    {
        return data[index];
    }

    const int& operator[](size_t index) const   // const 版本——用于只读访问
    {
        return data[index];
    }
};

VectorWrapper vw;
vw[0] = 100;          // 使用 non-const 的 []
cout << vw[0] << endl;
```

常量版本和非 const 版本的 `[]` 必须成对写——不然 const 对象就没法读元素了。

> 更经典的例子是自己实现一个 `String` 类，重载 `[]` 来访问某个位置的字符。你可以试着写一个 `MyString`，内部用 `char*` 存字符串，重载 `[]` 以返回某个位置的字符。

------

**`&` 取地址运算符**

重载 `&` 是合法的，但**极其少见**。C++ 允许你重载它来返回"别的什么东西"而不是真实地址，但这样做几乎必定坑到人——标准库的某些组件（如 `std::addressof`，用作获取对象真实地址的最后武器）正是为了防止被重载的 `&` 误导而设计的。

规律很简单：**别重载 `&`，除非你清楚自己在干什么，并且愿意挨骂。**

------

这三个运算符——`*`、`->`、`[]`——让你能打造"对外看起来完全不像自己本尊"的类型：智能指针看起来像普通指针，包装类看起来像它包裹的那个东西。这是 C++ 实现零开销抽象的重要工具，也是为什么很多 C++ 库能做到"用起来舒服、跑起来飞快"的秘诀之一。
