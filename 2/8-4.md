# 摄取与排放数据 · 文件输入输出

程序中的数据通常存在于内存中，程序一旦结束就会丢失。如果想要持久保存数据，就需要将其写入文件。反之，读取文件可以让程序获取之前保存的数据。

------

**文件流类**

C++ 提供了三个文件流类：

| 类 | 说明 |
| -- | ---- |
| `ifstream` | 输入文件流（读取文件） |
| `ofstream` | 输出文件流（写入文件） |
| `fstream` | 双向文件流（读写文件） |

使用前需要包含 `<fstream>` 头文件。

------

**写入文件**

```cpp
#include <fstream>
#include <iostream>
using namespace std;

int main()
{
    ofstream outFile("data.txt");  // 创建/打开文件
    
    if (!outFile)
    {
        cerr << "无法打开文件！" << endl;
        return 1;
    }
    
    // 写入数据（与 cout 用法相同）
    outFile << "Hello, File!" << endl;
    outFile << 42 << endl;
    outFile << 3.14 << endl;
    
    outFile.close();  // 关闭文件
    
    cout << "数据已写入文件" << endl;
    return 0;
}
```

------

**读取文件**

```cpp
#include <fstream>
#include <iostream>
#include <string>
using namespace std;

int main()
{
    ifstream inFile("data.txt");
    
    if (!inFile)
    {
        cerr << "无法打开文件！" << endl;
        return 1;
    }
    
    // 读取数据
    string line;
    int number;
    double decimal;
    
    getline(inFile, line);  // 读取一行
    inFile >> number;       // 读取整数
    inFile >> decimal;      // 读取浮点数
    
    cout << "字符串: " << line << endl;
    cout << "整数: " << number << endl;
    cout << "小数: " << decimal << endl;
    
    inFile.close();
    return 0;
}
```

------

**逐行读取**

```cpp
ifstream inFile("data.txt");
string line;

while (getline(inFile, line))
{
    cout << line << endl;

```

**文件打开模式**

```cpp
// 常用模式
ofstream out1("file.txt");                    // 默认，覆盖写入
ofstream out2("file.txt", ios::app);          // 追加写入
ofstream out3("file.txt", ios::binary);       // 二进制模式
fstream file("file.txt", ios::in | ios::out); // 读写模式

// 模式可以组合
fstream f("file.txt", ios::in | ios::out | ios::binary);
```

| 模式 | 说明 |
| ---- | ---- |
| `ios::in` | 读取 |
| `ios::out` | 写入 |
| `ios::app` | 追加到末尾 |
| `ios::trunc` | 清空文件（默认） |
| `ios::binary` | 二进制模式 |

------

**二进制文件**

```cpp
#include <fstream>
using namespace std;

struct Record
{
    int id;
    double value;
    char name[20];
};

int main()
{
    // 写入二进制数据
    Record r1 = {1, 3.14, "Test"};
    ofstream outFile("data.bin", ios::binary);
    outFile.write(reinterpret_cast<char*>(&r1), sizeof(r1));
    outFile.close();
    
    // 读取二进制数据
    Record r2;
    ifstream inFile("data.bin", ios::binary);
    inFile.read(reinterpret_cast<char*>(&r2), sizeof(r2));
    inFile.close();
    
    cout << r2.id << ", " << r2.value << ", " << r2.name << endl;
    
    return 0;
}
```

**文件位置**

```cpp
fstream file("data.txt", ios::in | ios::out);

// 获取当前位置
streampos pos = file.tellg();  // 读取位置
// streampos pos = file.tellp();  // 写入位置

// 移动到指定位置
file.seekg(0);                  // 移动到开头
file.seekg(10);                 // 移动到第 10 字节
file.seekg(0, ios::end);        // 移动到末尾
file.seekg(-5, ios::cur);       // 从当前位置后退 5 字节

// seekp 用于写入位置
file.seekp(0);
```

------

**检查文件状态**

```cpp
ifstream file("data.txt");

if (file.is_open())
    cout << "文件已打开" << endl;

if (file.eof())
    cout << "已到达文件末尾" << endl;

if (file.fail())
    cout << "操作失败" << endl;

if (file.bad())
    cout << "发生严重错误" << endl;

if (file.good())
    cout << "状态良好" << endl;
```

**完整示例：简单的学生成绩管理**

```cpp
#include <fstream>
#include <iostream>
#include <vector>
#include <string>
using namespace std;

struct Student
{
    string name;
    int score;
};

void saveStudents(const vector<Student>& students, const string& filename)
{
    ofstream file(filename);
    for (const auto& s : students)
    {
        file << s.name << " " << s.score << endl;
    }
}

vector<Student> loadStudents(const string& filename)
{
    vector<Student> students;
    ifstream file(filename);
    
    Student s;
    while (file >> s.name >> s.score)
    {
        students.push_back(s);
    }
    
    return students;
}

int main()
{
    // 保存数据
    vector<Student> students = {
        {"Alice", 95},
        {"Bob", 87},
        {"Carol", 92}
    };
    saveStudents(students, "students.txt");
    
    // 读取数据
    auto loaded = loadStudents("students.txt");
    for (const auto& s : loaded)
    {
        cout << s.name << ": " << s.score << endl;
    }
    
    return 0;
}
```
