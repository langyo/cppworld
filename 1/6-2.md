# 可供理解的数据 · 字符串库

在计算机的世界里，文字是以字符的形式存储的。我们已经见过 C 风格的字符串——用字符数组来存储：

```cpp
char str[] = "Hello";
```

但这种方式有很多不便之处：需要手动管理内存、容易越界、操作繁琐……

C++ 标准库提供了 `string` 类，让字符串的处理变得简单而安全。

------

**创建字符串**

使用 `string` 类需要包含 `<string>` 头文件：

```cpp
#include <iostream>
#include <string>
using namespace std;

int main()
{
    string s1;                    // 空字符串
    string s2 = "Hello";          // 用字符串字面量初始化
    string s3("World");           // 另一种初始化方式
    string s4(5, 'A');            // 5 个 'A'，即 "AAAAA"
    string s5 = s2;               // 复制 s2
    string s6 = s2 + " " + s3;    // 字符串拼接
    
    cout << s6 << endl;  // 输出: Hello World
    
    return 0;
}
```

**字符串的基本操作**

```cpp
string s = "Hello, World!";

// 获取长度
cout << s.length() << endl;   // 13
cout << s.size() << endl;     // 13，与 length() 相同

// 判断是否为空
if (s.empty())
    cout << "字符串为空" << endl;
else
    cout << "字符串不为空" << endl;

// 访问单个字符
cout << s[0] << endl;         // H
cout << s.at(1) << endl;      // e（带边界检查）

// 修改单个字符
s[0] = 'h';
cout << s << endl;            // hello, World!
```

`at()` 方法和下标运算符 `[]` 都可以访问字符，区别在于 `at()` 会进行边界检查，越界时会抛出异常。

------

**字符串的拼接**

拼接字符串非常简单，直接用 `+` 运算符即可：

```cpp
string first = "Hello";
string second = "World";

string result = first + ", " + second + "!";
cout << result << endl;  // Hello, World!

// 也可以用 += 追加
first += "!!!";
cout << first << endl;   // Hello!!!

// append 方法
string s;
s.append("Hello");
s.append(" ");
s.append("World");
cout << s << endl;       // Hello World
```

------

**子字符串操作**

```cpp
string s = "Hello, World!";

// 获取子字符串
string sub = s.substr(0, 5);    // 从位置 0 开始，取 5 个字符
cout << sub << endl;            // Hello

// 查找子字符串
size_t pos = s.find("World");   // 返回首次出现的位置
if (pos != string::npos)
    cout << "找到了，位置: " << pos << endl;  // 位置: 7
else
    cout << "没找到" << endl;

// 从后往前找
pos = s.rfind('o');
cout << "最后一个 'o' 的位置: " << pos << endl;  // 8

// 替换
string str = "Hello, World!";
str.replace(7, 5, "C++");       // 从位置 7 开始，替换 5 个字符
cout << str << endl;            // Hello, C++!

// 删除
str.erase(5, 2);                // 从位置 5 开始，删除 2 个字符
cout << str << endl;            // HelloC++!

// 插入
str.insert(5, ", ");            // 在位置 5 插入
cout << str << endl;            // Hello, C++!
```

------

**字符串比较**

字符串可以直接用比较运算符进行比较，按字典序（ASCII 码顺序）：

```cpp
string a = "apple";
string b = "banana";
string c = "apple";

if (a == c)
    cout << "a 和 c 相等" << endl;

if (a < b)
    cout << "apple 在 banana 之前" << endl;

// compare 方法
int result = a.compare(b);
if (result < 0)
    cout << "a < b" << endl;
else if (result > 0)
    cout << "a > b" << endl;
else
    cout << "a == b" << endl;
```

------

**字符串与数字的转换**

C++11 提供了方便的转换函数：

```cpp
#include <string>

// 数字转字符串
int num = 42;
string s = to_string(num);
cout << s << endl;  // "42"

double pi = 3.14159;
string piStr = to_string(pi);
cout << piStr << endl;  // "3.141590"

// 字符串转数字
string numStr = "123";
int n = stoi(numStr);         // string to int
cout << n + 1 << endl;        // 124

string floatStr = "3.14";
double d = stod(floatStr);    // string to double
cout << d * 2 << endl;        // 6.28

// 其他转换函数：stol, stoll, stof, stold 等
```

**输入输出**

```cpp
string name;

// 读取单个词（遇到空白字符停止）
cout << "请输入你的名字: ";
cin >> name;
cout << "你好, " << name << "!" << endl;

// 读取整行（包括空格）
cin.ignore();  // 忽略之前输入留下的换行符
cout << "请输入一句话: ";
getline(cin, name);
cout << "你说: " << name << endl;
```

`cin >> s` 只能读取一个词，遇到空格、制表符或换行符就停止。如果需要读取整行，应该使用 `getline()`。

------

**遍历字符串**

```cpp
string s = "Hello";

// 方式一：下标遍历
for (size_t i = 0; i < s.length(); i++)
{
    cout << s[i] << " ";
}
cout << endl;

// 方式二：范围 for 循环
for (char c : s)
{
    cout << c << " ";
}
cout << endl;

// 方式三：迭代器
for (string::iterator it = s.begin(); it != s.end(); ++it)
{
    cout << *it << " ";
}
cout << endl;

// C++11 简化写法
for (auto it = s.begin(); it != s.end(); ++it)
{
    cout << *it << " ";
}
cout << endl;
```

------

**实用示例：统计单词数量**

```cpp
#include <iostream>
#include <string>
#include <sstream>
using namespace std;

int main()
{
    string text = "Hello World This is a test";
    
    // 方法一：使用字符串流
    istringstream iss(text);
    string word;
    int count = 0;
    
    while (iss >> word)
    {
        count++;
        cout << "单词 " << count << ": " << word << endl;
    }
    
    cout << "共 " << count << " 个单词" << endl;
    
    return 0;
}
```

------

**C 风格字符串与 string 的转换**

有时我们需要在两者之间转换：

```cpp
// C 风格字符串转 string
const char* cstr = "Hello";
string s = cstr;  // 自动转换

// string 转 C 风格字符串
string str = "World";
const char* ptr = str.c_str();  // 获取 C 风格字符串指针
```

`c_str()` 返回一个以空字符结尾的 C 风格字符串指针，常用于调用需要 C 风格字符串的函数。
