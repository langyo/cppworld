# 自助服务 · 默认实参与函数指针

在餐厅点餐时，如果你不特别说明，服务员可能会给你提供标准的配菜和调料。但如果你有特殊要求，也可以自己指定。

C++ 函数的**默认实参**就是这样一种机制——你可以为参数提供一个默认值，调用时如果不提供该参数，就使用默认值。

------

**默认实参**

来看一个简单的例子：

```cpp
void greet(string name = "朋友")
{
    cout << "你好, " << name << "!" << endl;
}

int main()
{
    greet("小明");   // 输出: 你好, 小明!
    greet();         // 输出: 你好, 朋友!（使用默认值）
    return 0;
}
```

定义默认实参时，只需在参数后面加上 `= 默认值` 即可。

**规则一：默认实参必须从右往左连续提供**

```cpp
// 正确
void func1(int a, int b = 10, int c = 20);

// 错误！不能跳过中间的参数
void func2(int a = 10, int b, int c = 20);  // 编译错误

// 正确
void func3(int a, int b, int c = 20);

// 正确
void func4(int a = 1, int b = 2, int c = 3);
```

**规则二：调用时省略的参数也必须从右往左连续**

```cpp
void display(int a, int b = 2, int c = 3, int d = 4);

display(1);           // a=1, b=2, c=3, d=4
display(1, 5);        // a=1, b=5, c=3, d=4
display(1, 5, 6);     // a=1, b=5, c=6, d=4
display(1, 5, 6, 7);  // a=1, b=5, c=6, d=7

// 错误！不能跳过中间的参数
// display(1, , 6);   // 编译错误
```

**规则三：默认值只能在声明中指定一次**

```cpp
// 头文件中的声明（指定默认值）
void func(int a, int b = 10);

// 源文件中的定义（不再指定默认值）
void func(int a, int b)
{
    // ...
}
```

通常，默认值放在函数声明中（头文件里），而不是定义中（源文件里）。

------

**默认实参的实际应用**

默认实参在实际开发中非常有用，可以简化函数调用：

```cpp
// 日志函数，默认级别为 INFO
void log(const string& message, int level = 1)
{
    string levelStr;
    switch (level)
    {
        case 0: levelStr = "DEBUG"; break;
        case 1: levelStr = "INFO"; break;
        case 2: levelStr = "WARNING"; break;
        case 3: levelStr = "ERROR"; break;
    }
    cout << "[" << levelStr << "] " << message << endl;
}

int main()
{
    log("程序启动");              // 默认 INFO 级别
    log("检测到异常", 2);         // WARNING 级别
    log("致命错误", 3);           // ERROR 级别
    return 0;
}
```

------

**函数指针**

在 C++ 中，函数也是一种数据，可以被存储和传递。**函数指针**就是指向函数的指针。

函数指针的声明语法：

```cpp
返回类型 (*指针名)(参数类型列表);
```

看一个例子：

```cpp
int add(int a, int b)
{
    return a + b;
}

int subtract(int a, int b)
{
    return a - b;
}

int main()
{
    // 声明一个函数指针
    int (*operation)(int, int);
    
    // 将函数地址赋给指针
    operation = add;
    cout << operation(5, 3) << endl;  // 输出 8
    
    operation = subtract;
    cout << operation(5, 3) << endl;  // 输出 2
    
    return 0;
}
```

函数名本身就代表函数的地址，所以 `operation = add` 和 `operation = &add` 是等价的。

------

**函数指针作为参数**

函数指针的一个重要用途是作为参数传递给其他函数，实现**回调**机制：

```cpp
#include <iostream>
using namespace std;

// 两个运算函数
int add(int a, int b) { return a + b; }
int multiply(int a, int b) { return a * b; }

// 接受函数指针作为参数
void compute(int x, int y, int (*func)(int, int))
{
    cout << "结果: " << func(x, y) << endl;
}

int main()
{
    compute(5, 3, add);       // 输出: 结果: 8
    compute(5, 3, multiply);  // 输出: 结果: 15
    return 0;
}
```

这种模式允许我们编写更加灵活和通用的代码。

------

**用 typedef 简化函数指针**

函数指针的语法比较复杂，可以用 `typedef` 或 `using` 来简化：

```cpp
// 使用 typedef
typedef int (*Operation)(int, int);

// 或使用 using（C++11）
using Operation = int (*)(int, int);

// 现在可以这样使用
Operation op = add;
cout << op(5, 3) << endl;

void compute(int x, int y, Operation func)
{
    cout << func(x, y) << endl;
}
```

------

**函数指针数组**

我们甚至可以创建函数指针的数组：

```cpp
int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }
int mul(int a, int b) { return a * b; }
int div(int a, int b) { return a / b; }

int main()
{
    // 函数指针数组
    int (*ops[])(int, int) = {add, sub, mul, div};
    const char* opNames[] = {"+", "-", "*", "/"};
    
    int a = 20, b = 5;
    for (int i = 0; i < 4; i++)
    {
        cout << a << " " << opNames[i] << " " << b << " = " << ops[i](a, b) << endl;
    }
    
    return 0;
}
```

输出：

```
20 + 5 = 25
20 - 5 = 15
20 * 5 = 100
20 / 5 = 4
```

------

**返回函数指针的函数**

函数也可以返回函数指针：

```cpp
// 声明起来比较复杂
int (*getOperation(char op))(int, int)
{
    switch (op)
    {
        case '+': return add;
        case '-': return sub;
        case '*': return mul;
        case '/': return div;
        default: return nullptr;
    }
}

// 使用 typedef 会更清晰
typedef int (*Operation)(int, int);

Operation getOperation(char op)
{
    switch (op)
    {
        case '+': return add;
        case '-': return sub;
        case '*': return mul;
        case '/': return div;
        default: return nullptr;
    }
}
```

------

在现代 C++ 中，我们通常会使用更高级的特性（如 lambda 表达式和 `std::function`）来替代原始的函数指针，但理解函数指针的概念对于理解这些高级特性非常有帮助。
